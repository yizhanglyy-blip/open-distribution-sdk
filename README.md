# open-distribution-sdk


## 接入说明

   为了帮助分销商快速接入我们的分销系统，我们提供了sdk帮助分销商理解GDS接口文档中的api，对GDS接口文档中的api给出了调用示例，并提供了一些基础的下单场景的请求传参示例、以及回调推送处理示例。

## 模块介绍
  分销商在接口测试过程中，参考以下的Demo文件：
- [1] ProductDemo.java 对应GDS接口文档Part E 产品信息相关接口
- [2] OrderDemo.java 对应GDS接口文档Part F 预订相关接口
- [3] CategoryDemo.java 对应GDS接口文档Part G 分类信息接口
- [4] PushInformationDemo.java 对应GDS接口文档Part J 信息推送
- [5] AccountParams.java 对应分销商账号信息，作为调用接口的前置参数
